## Description

This module shows HTTP Headers returned by the scanned systems.

## Verification Steps

1. Do: ```use auxiliary/scanner/http/http_header```
2. Do: ```set RHOSTS [IP]```
3. Do: ```set RPORT [PORT]```
4. Do: ```run```
