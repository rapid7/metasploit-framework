## Description

This module allows you to perform SQL queries against a database using known-good credentials.

## Verification Steps

1. Do: ```use auxiliary/admin/mssql/mssql_sql```
2. Do: ```set PASSWORD [password1]```
3. Do: ```set RHOSTS [IP]```
4. Do: ```set [SQL Command]```
5. Do: ```run```

## Scenarios

